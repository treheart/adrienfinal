Quiz Paths

This folder contains a structured `quiz.json` and result pages used to power the Adrien Ninet diagnostic.

Files:
- quiz.json: questions, paths, and result mappings
- results/*.md: result pages for each path (Logiaweb, Klime Studio, Limora AI)

Usage:
Import or read `content/quiz-paths/quiz.json` in your app and route users to the matching markdown result file based on the selected path.
